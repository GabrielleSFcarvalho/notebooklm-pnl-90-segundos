##  Contexto e Objetivos

Como profissional de tecnologia, entendo que **código não roda sozinho**. Grandes projetos nascem de grandes colaborações. Meu objetivo com este repositório é demonstrar a fusão entre **Hard Skills (IA Generativa)** e **Soft Skills (PNL)**.

Utilizei o **Google NotebookLM** não apenas para resumir, mas para criar um **Gêmeo Digital de Conhecimento**, transformando a teoria de Nicholas Boothman em um mentor consultivo para conexões humanas instantâneas, seja no ambiente profissional ou social.

##  Miniguia de Estudo (Entrega Final)

### 1. Resumo Estruturado: O Algoritmo da Conexão
* **Atitude:** É o "software" que você roda antes de abrir a boca. Atitude entusiasta gera conexão; atitude defensiva gera ruído.
* **Rapport:** Sincronização de linguagem corporal e tom de voz (espelhamento).
* **VAK (Visual, Auditivo, Cinestésico):** Identificar o canal de preferência do outro para transmitir a mensagem sem interferências.

### 2. Glossário de Conceitos Aprendidos

* PNL (Programação Neurolinguística): Estudo de como a linguagem (verbal e não verbal) programa nosso cérebro e influencia nossos comportamentos e conexões.

* Rapport: Técnica de remover as barreiras instintivas de "lutar ou fugir" para criar uma conexão humana fluida e sem esforço.

* Sistema VAK (Visual, Auditivo, Cinestésico): Identificação da "frequência" sensorial preferida de uma pessoa para adaptar a nossa comunicação à dela.

* Sincronização Inconsciente (Camaleão Social): Ajuste sutil de postura e tom de voz para que o interlocutor sinta, subconscientemente, que somos parecidos e confiáveis.

* Modelagem: Identificar e replicar padrões de pensamento e linguagem de comunicadores de sucesso.

### 3. Ferramentas para Revisão (Prompts Reutilizáveis)
* Para aprofundar conceitos: "Nicholas, explique a diferença entre uma pessoa Visual e uma Cinestésica usando exemplos de como elas se comportam em uma reunião."

* Para superar desafios pessoais: "Sou uma pessoa tímida e preciso fazer networking em um evento de tecnologia. Quais são os 3 primeiros passos que devo dar nos primeiros 90 segundos?"

* Para análise de progresso: "Com base nas fontes, como posso saber se consegui estabelecer um Rapport genuíno com alguém? Quais sinais devo observar?"
